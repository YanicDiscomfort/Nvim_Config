# NvChad config
> this Repo contains my personal config for [NvChad](https://nvchad.com/)

if you want to use them:

1. **install dependencies**
    ```sh
    sudo pacman -S nerd-fonts ripgrep lua make gcc neovim nodejs npm silicon
    ```

2. **Backup your current config, if you have one**
    ```sh
    mv ~/.config/nvim/ ~/.config/old_nvim_config/
    ```
3. **get config**
    ```sh
    git clone https://github.com/YanicDiscomfort/Nvim_Config ~/.config/nvim/ && nvim
    ```
4. **READ THE DOCUMENTATIONS**
    [NvChad-Docs](https://nvchad.com/docs/quickstart/install)
